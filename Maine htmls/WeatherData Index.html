For this project we'll be creating a visualization dashboard website using visualizations we've created as part of API-Challenge assignment. Specifically, we'll be plotting weather data of more than 500 cities across the world of varying distance from the equator with random latitude and longitude coordinates ( Weather project). The weather data was queried from OpenWeatherMap API, plots has been made Finally the results about the dependence of weather on the latitude value has been analyzed.
